Sobre

Sou uma Desenvolvedora Java apaixonada por tecnologia e inovação. Graduada em Análise e Desenvolvimento de Sistemas pela Faculdade Impacta, possuo também a certificação AWS Cloud Practitioner.

Atualmente, trabalho na RGC Consultoria Engenharia, onde sou responsável pela manutenção e implementação de novas funcionalidades em sistemas baseados em Java e Postgres.

Minha experiência inclui o desenvolvimento de Backends para Frontends (BFFs) com Spring Boot e a construção de microserviços que atendem às necessidades dos usuários. Durante minha passagem pelo Itaú Unibanco, atuei no projeto "Contextualização Mercado", aprimorando a observabilidade do sistema com ferramentas como Prometheus e Grafana.

Além disso, no programa Quero Ser Dev da Raia Drogasil, tive a oportunidade de aprimorar minhas habilidades Full Stack, desenvolvendo aplicações com Java e Angular.

Estou sempre em busca de novos desafios e oportunidades para aplicar e expandir meus conhecimentos técnicos. Acredito que cada projeto é uma oportunidade para aprender e contribuir de forma significativa.
